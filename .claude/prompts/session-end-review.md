## Session End: Documentation Review Protocol

Voordat deze sessie eindigt, doorloop dit protocol.

---

### Stap 1: Update HISTORY.md

1. **Vat samen wat er is gedaan** (taken, commits)
2. **Log key decisions** met rationale
3. **Noteer learnings** uit deze sessie
4. **Schrijf "Next Session" acties**

### Stap 2: Check CLAUDE.md Candidates

Scan deze sessie voor potentiele CLAUDE.md updates:

| Categorie | Check |
|-----------|-------|
| Nieuwe coding patterns | [ ] |
| Project conventions verduidelijkt | [ ] |
| Tool/dependency preferences | [ ] |
| Workflow optimalisaties | [ ] |
| Fouten die voorkomen moeten worden | [ ] |

### Stap 3: 5-Check voor Elke Candidate

| Candidate | H | S | G | N | A | Score |
|-----------|---|---|---|---|---|-------|
| "..." | Y/N | Y/N | Y/N | Y/N | Y/N | X/5 |

- **H**erhaalbaarheid: Relevant voor toekomstige sessies?
- **S**tabiliteit: Stabiel, niet experimenteel?
- **G**enericiteit: Geldt voor hele project?
- **N**on-redundantie: Niet al in CLAUDE.md?
- **A**ctionability: Direct bruikbaar?

### Stap 4: Acties

- **Score 5/5**: Voeg toe aan CLAUDE.md
- **Score <5**: Log in HISTORY.md waarom NIET toegevoegd

### Stap 5: Handover Note

Schrijf in HISTORY.md een "Next Session" sectie:
- Context voor volgende sessie
- Immediate acties
- Eventuele blockers

---

**Commit alle documentation updates voordat je afsluit!**
